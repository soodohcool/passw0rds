Help

  • Number of Passphrases: Specifies the total number of passphrases to generate.
  • Leet Characters: Sets the minimum and maximum range of characters to transform within each passphrase.
    • Min: The minimum number of characters that will be transformed.
    • Max: The maximum number of characters that will be transformed.
    The actual number of transformed characters will be between these two values.
  • Min Word Length: Sets the minimum length for each word fetched from the word files.
  • Max Word Length: Sets the maximum length for each word fetched from the word files.
  • Pattern: Determines the sequence and types of words to be used in the passphrase. You can employ various combinations such as 'AVN', 'AAVNPAP', etc.
    • A: Adjective
    • V: Verb
    • N: Noun
    • P: Plural Nouns
  • Transform: Selects the transformation method applied to the passphrase:
    • Leet: Applies full leet speak transformations.
    • Mini Leet: Applies a simplified version of leet speak.
    • Plain: No transformation is applied.
